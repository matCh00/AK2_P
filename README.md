# AK2_P

## Kalkulator wielofunkcyjny liczb zmiennoprzecinkowych `do 80 bitów`

+ funkcje:
  - dodawanie
  - odejmowanie
  - mnożenie
  - dzielenie
  - pierwiastek
  - potęga
  - sinus
  - cosinus
  - tangens
  - cotangens
  - równanie kwadratowe
  - ciąg Fibonacciego

+ sposoby zaokrąglania:
  - obcięcie
  - w górę
  - w dół
  - do najbliższej
